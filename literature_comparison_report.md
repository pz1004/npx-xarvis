# 문헌 조사 및 관련 연구 비교 보고서

## 1. 개요

본 보고서는 제안된 알고리즘(Event-Domain Denoising for Direct SNN Ingestion with Split-State Semantics)과 관련된 최신 연구를 8개 주제 영역에 걸쳐 조사한 결과를 종합합니다.

---

## 2. 관련 연구 분류 및 비교

|------|------|-----------|------|----------------|

## 3. 제안 방법의 차별점 요약

제안된 방법은 다음과 같은 고유한 위치를 차지합니다:

1. **분리된 상태 의미론:** 기존 어떤 방법도 원시 상태와 수락된 상태를 명시적으로 분리하여 극성 조건부 임계값의 비중복성을 수학적으로 증명하지 않았습니다.

2. **이론-구현-평가 통합:** 수학적 분석(BA 고정점 경계, 샷 노이즈 제거 비율)과 하드웨어 인식 설계, 그리고 재현 가능한 평가 프로토콜을 하나의 프레임워크에 통합합니다.

3. **신뢰도 코딩을 통한 SNN 직접 연동:** 기존 방법들은 노이즈 제거와 SNN 추론을 별개의 단계로 취급하지만, 제안 방법은 신뢰도 정보를 SNN 입력 가중치로 직접 전달합니다.

---

## 4. 제안 방법의 약점 (문헌 대비)

1. **정확도 벤치마크 부재:** 현재 논문은 이론 및 아키텍처 기여만 제시하며, SOTA SNN들(SpiLiFormer: CIFAR10-DVS 87.3%, SpikePoint: DVS Gesture 98.5%)과의 실증적 비교가 없습니다.

2. **처리 속도:** 순수 Python 구현은 PCLF(60ns/이벤트)나 MLPF(10ns/이벤트)와 비교할 수 없으며, Numba/FPGA 최적화가 필수적입니다.

3. **학습 기반 방법 대비 적응성:** EDmamba(AUC 0.9944)나 WedNet 같은 학습 기반 방법은 다양한 노이즈 환경에 적응할 수 있지만, 제안 방법은 고정된 파라미터에 의존합니다.

4. **평가 메트릭 제한:** ESR, AOCC 같은 최신 구조적 보존 메트릭을 사용하지 않습니다.

---

## 5. 개선을 위한 핵심 시사점

문헌 조사를 통해 도출된 주요 개선 방향:

1. **Numba JIT 최적화:** 필터 처리 속도를 수십~수백 배 향상시켜 대규모 실험 가능하게 함
2. **적응형 임계값:** 지역 이벤트 밀도에 따라 파라미터를 동적으로 조정
3. **더 강력한 SNN 백본:** 시간적 정규화(Temporal Regularization)나 측면 억제(Lateral Inhibition) 통합
4. **추가 평가 메트릭:** ESR, AOCC 등 구조적 보존 메트릭 추가
5. **실제 노이즈 데이터셋:** LED 데이터셋 등을 활용한 실제 환경 검증
6. **차분 코딩(Differential Coding):** 스파이크 수 감소를 통한 에너지 효율 개선
